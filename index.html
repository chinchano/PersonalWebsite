My Work

Password Validation Form

This is a password validation page I am building using jQuery for the behavior side of the project. Some CSS styling used for the input boxes and to make the submit button look presentable. Currently, I'm at the stage where the warning boxes remove once the correct amount of characters are entered using "keyup" and "focus" from the jQuery library. The Submit button will not work if a username isn't entered as well. Now, I'm working on having an alert pop up to the user when something from the form is missing.

Technologies used-jQuery, HTML5, CSS3

Digital Alarm Clock

This project I'm pretty proud of. I learned a lot about the functions that are already available with Javascript. I never knew about the Date() function. Using that function I was able to create variables for hours, minutes and seconds. Using those variables I was able to construct a string that resembles a digital clock. I also leared about the setInterval() function when I couldn't figure out why the clock wouldn't tick. Now I'm working on an alarm function that is fired when the submit button I created is pressed

Technologies used-Javascript, HTML5, CSS3
